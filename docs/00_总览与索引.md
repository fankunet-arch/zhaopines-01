## 这是什么

一个面向**西班牙华人低端服务业**（餐厅厨师/跑堂、百元店、美甲店等）的**极简信息发布型招聘平台**。形态接近"分类信息墙"，核心是"30 秒贴一条信息就走"，不做简历库、在线沟通等重功能。

## 关键决策速查表

| 维度 | 决策 |
|------|------|
| 目标用户 | 西班牙华人低端服务业（厨师、跑堂、百元店、美甲等） |
| 产品形态 | 极简信息墙；纯中文界面 + 底部西语合规声明；移动优先 |
| 三端 | 前台 / 用户后台 / 管理员后台 |
| 信息类型 | 招聘、求职两类 |
| 必填字段 | 内容、联系人/姓名、电话、地区、类别；微信选填 |
| 不做 | 简历、在线沟通(IM)、薪资/工时、智能匹配 |
| 发布权限 | 游客可发但不可改；注册用户可改/删/顶帖（"要控制权就注册"） |
| 游客帖 | 发出即不可改、不可认领；只能等到期或被管理员删 |
| 有效期 | 30 天到期下架（数据保留）+ 保底最新 N 条防墙空（招聘/求职各算） |
| 重复发布 | 同电话同内容每日限 1 条；7 天内顶帖代替新发 |
| 排序 | 置顶 > 时间倒序（论坛式） |
| 类别 | 固定下拉 + 可录入 → 待审核表隔离 → 审核通过并入正式表 |
| 地区 | 固定两级下拉（大区/市），覆盖全西班牙 |
| 登录 | 仅 Google（零成本）；无账号者走游客。管理员也走 Google + 白名单 |
| 密码 | **全站零密码**（普通用户与管理员均不存密码） |
| 举报 | 任何人可举报；触发管理员邮件（去重 + 合并降频，防刷爆） |
| 邮件发送 | Brevo 事务邮件免费档 + 域名发件地址 |
| 管理后台入口 | **约定为 `https://<域名>/c/cp/`**（对应 `zp_html/c/cp/`），不用 /admin 之类可猜路径 |
| 管理员分级 | **超级管理员**（全部功能+参数配置+管理员管理+置顶券恒可）/ **普通管理员**（信息/类别/举报/失效；置顶券权限由超管开关 `normal_admin_can_coupon` 控制）；不能停用/降级自己防锁死 |
| 置顶券 | 兑换码模式（管理员生成 → 用户兑换 → 用于某帖）；有台账可作废重发；已使用不可作废；位满拒绝时券保留；默认不防转赠。**已实装（原 P2 提前）** |
| 置顶位上限 | 后台可调；"准入闸"逻辑——调小不踢存量，自然回落 |
| URL 形态 | **无扩展名**（/publish、/detail?id=随机码）；显式 .php 与 index 一律 301 归一（/index→/、/c/cp/index→/c/cp/），Apache .htaccess / Nginx 重写实现 |
| 字体 | Bricolage Grotesque + Noto Sans SC **本地托管**（assets/fonts/，保留 unicode-range 子集按需加载）；不连 Google Fonts（GDPR + 加载速度） |
| 错误输出 | 对外零内部信息：display_errors 关闭，未捕获异常/配置缺失/视图缺失一律输出通用文案，细节只进服务器日志；页面输出中不得出现表名/文件路径 |
| 防机器人 | 验证码 + 蜜罐 + 频率限制；不人工前置审核，可疑仅后台标记 |
| 防爬虫 | 联系方式点击才取号（API+频率限制）+ 字段名混淆可轮换；**定位：尽力而为，非杜绝** |
| 失效治理 | 浏览者可标记"已失效"；注册用户可主动下架 |
| 查看次数 | 取号时电话/微信**分开计数**，前台公开显示热度；同IP同帖同类型短时去重防刷 |
| 安全 | 内部主键自增不外露，对外用不可枚举随机码；机密放 app/config 不入库 |
| 合规 | 隐私政策 + Cookie（GDPR）；个人数据最小化收集 |
| 冷启动 | **待定**（建议地推/商家授权首发，不建议采集他人个人数据） |

## 仍待你最终决定的事项（不阻塞动工）

- **冷启动数据来源**：上线时墙不能空。建议真实地推/商家授权首发。
- 尚未落定的技术细节：求职帖结构化数据方案、中文搜索是否上 FULLTEXT、外键约束范围。

**v1.1 已落定的技术细节**：验证码形态=算术题（Session 存答案）｜登录态承载=PHP 原生
Session（Cookie HttpOnly+SameSite=Lax）｜随机码算法=去易混字符（0O1lI）的 54 字符表随机
（信息 10 位 / 用户 8 位 / 兑换码 ZP+10 位）｜置顶到期回落=读取时惰性判断（无定时任务）｜
内容限长 1000 字｜手动顶帖同帖 24 小时限一次｜各类频率阈值入 settings 表后台可调。
- **隐私政策占位项**：法定名称、地址、联系邮箱等，上线前填齐并经律师审阅。

## 重要定位声明（贯穿全套）

- **防护是"尽力而为"，非"安全保证"**：点击取号、字段名混淆、验证码、蜜罐等手段旨在**提高骚扰/采集成本**，不承诺杜绝被爬或被刷。
- **本套文档不含应用层业务代码**；数据库 DDL 与种子 SQL 属于可执行的结构/数据脚本。
